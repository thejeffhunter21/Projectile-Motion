This is a projectile motion simulator. My goal for this project was to familiarize myself with applying learned knowledge in my classes and seeing how I can translate them to code. I used HTML/CSS, C++, and JavaScript to make this. Emscripten was used to translate C++ to .WASM, which is how I can host the project to my browser. I got inspiration from PhET Lab's simulator. I also wanted to get used to using git.
